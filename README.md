# 사용한도 성분 퀴즈 — 맞춤형화장품 조제관리사

화장품 안전기준 등에 관한 규정 별표2(보존제·자외선차단제·염모제·기타 성분 사용한도)를
암기용 사지선다 퀴즈로 만든 웹앱입니다. `index.html` 파일 하나로 동작합니다.

## 사용 방법 (그냥 열어보기)
`index.html`을 더블클릭하거나 브라우저로 열면 바로 실행됩니다. 서버나 설치가 필요 없습니다.

## GitHub에 올려서 링크로 보기 (GitHub Pages)
1. GitHub에서 새 저장소(repository)를 만듭니다. (예: `cosmetic-quiz`)
2. `index.html`과 `reference.md`를 저장소 루트에 업로드합니다.
3. 저장소의 **Settings → Pages**로 이동합니다.
4. **Source**를 `Deploy from a branch`로 두고, 브랜치를 `main`, 폴더를 `/(root)`로 선택 후 저장합니다.
5. 1~2분 후 `https://[깃허브아이디].github.io/[저장소이름]/` 주소로 접속하면 퀴즈가 바로 뜹니다.

## 파일 구성
- `index.html` — 퀴즈 앱 (사지선다 · 정답 즉시 공개형 · 헷갈리는 항목 비교표 포함)
- `reference.md` — 원본 표 전체를 정리한 암기 참고 문서 (퀴즈에 없는 항목까지 전부 포함)

## 퀴즈 구성 방식
- **카테고리**: 보존제 / 자외선차단 / 염모제 / 기타성분 — 상단 칩(chip)으로 필터링 가능
- **정답 공개 방식**: 문제를 고르면 그 즉시 정답·오답이 표시되고, 헷갈리기 쉬운 유사 항목들을
  같은 화면의 비교표에서 함께 보여줍니다. (실전 모의고사보다 암기 교정에 최적화된 방식)
- 문제는 매 회차 랜덤 순서로 섞여서 출제됩니다.

## 문제를 더 추가하고 싶다면
`index.html` 안의 `QUESTIONS` 배열에 아래 형식으로 객체를 추가하면 됩니다.

```js
{cat:"카테고리명", q:"문제 텍스트 (HTML 가능)",
 opts:["보기1","보기2","보기3","보기4"], answer:0, // 정답 인덱스(0부터 시작)
 explain:"정답 해설",
 compare:[["비교 항목명","수치"], ["비교 항목명2","수치2"]]}
```
