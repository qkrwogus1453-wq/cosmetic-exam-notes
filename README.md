# 사용한도 성분 퀴즈 — 맞춤형화장품 조제관리사

화장품 안전기준 등에 관한 규정 별표2(보존제·자외선차단제·염모제·기타 성분 사용한도)를
암기용 사지선다 퀴즈로 만든 웹앱입니다. **총 258문항**.

## 파일 구성
- `build.py` — **원본 소스.** 성분 데이터를 여기서 관리하고, 실행하면 `index.html`이 생성됩니다.
- `index.html` — 빌드된 퀴즈 앱 (이 파일 하나만 있으면 브라우저에서 바로 실행)
- `reference.md` — 원본 표 전체 참고 문서

## 퀴즈 특징
- **보기 자동 셔플** — 문제를 열 때마다 A/B/C/D 순서가 무작위로 바뀝니다. 위치로 답을 외울 수 없어요.
- **헷갈리는 오답** — 오답 보기는 아무 숫자가 아니라, **같은 카테고리에서 수치가 가장 가까운 다른 성분의 실제 한도**입니다.
- **문제 유형 4종**
  - `한도` — "X의 사용한도는?" (165문항)
  - `조건부` — "X를 [특정 조건]에 사용할 때 한도는?" (71문항)
  - `부정형` — "잘못 연결된 것은?" (5문항)
  - `복합` — 순서 나열, 묶음 판별, 최댓값/최솟값 찾기 등 (17문항)
- **즉시 정답 공개 + 비교표** — 답을 고르면 바로 정답/오답이 뜨고, 헷갈리기 쉬운 유사 항목 수치를 표로 함께 보여줍니다.
- **출제 문항 수 선택** — 10 / 20 / 30 / 전체 중 선택
- **오답만 다시 풀기** — 결과 화면에서 틀린 문제만 재시도 가능
- **연속 정답 표시** — 3문제 이상 연속 맞히면 스트릭 표시

## 실행 방법
`index.html`을 브라우저로 열면 끝입니다. 설치·서버 불필요.

## GitHub Pages 배포
1. 저장소에 `index.html`을 업로드 (`build.py`, `reference.md`도 함께 두면 좋습니다)
2. **Settings → Pages → Source: Deploy from a branch → main / (root) → Save**
3. `https://[아이디].github.io/[저장소이름]/` 로 접속

## 문제 수정·추가 방법
`index.html`을 직접 고치지 말고 **`build.py`를 고친 뒤 다시 빌드**하세요.

```bash
python3 build.py    # index.html이 새로 생성됩니다
```

- **단일 한도 성분 추가** → `SIMPLE` 리스트에 `("카테고리", "성분명", "한도", "비고")` 추가
- **조건별로 한도가 갈리는 성분** → `CONDITIONAL` 리스트에 추가 (조건마다 문제가 1개씩 자동 생성됨)
- **부정형·복합형 문제 직접 작성** → `HANDWRITTEN` 리스트에 추가

오답 보기와 비교표는 데이터에서 자동으로 만들어지므로, 성분만 넣으면 문제는 알아서 생성됩니다.
